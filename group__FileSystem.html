◆ ModeFlags

enum ModeFlags
strong

Flags to indicate access modes.

  • READ : Read-only access
  • WRITE : Write-only access
  • READ_WRITE : Read + Write access (read first, then write)
  • WRITE_READ : Write + Read access (write first, then read)
